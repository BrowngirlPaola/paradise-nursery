# Paradise Nursery

## Project Name
Paradise Nursery - An online houseplant shopping application.

## Author
BrowngirlPaola

## Description
Paradise Nursery is a React and Redux shopping app where users can browse houseplants, add them to a cart, and manage their purchases.

## Features
- Browse plants by category
- Add plants to shopping cart
- Update quantities
- Remove items from cart
- View total cost
